# trnql

## Installation

trnql is available through [CocoaPods](http://cocoapods.org). Install trnql by adding it to your Podfile:

```ruby
platform :ios, '8.0'
use_frameworks!

target 'MyApp' do
	pod 'trnql'
end
```
